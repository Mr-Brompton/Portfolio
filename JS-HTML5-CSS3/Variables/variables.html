Variables

What are Variables?

We can think of variables as a 'name' for our peices of data, we refer to these names as 'Identifiers'. We must give our data an identifier in order to use it later.

Any type of data can be stored in a variable. If we pose "The question of life, the universe and everything?", The string could be saved as the variable(var) "myQuestion"

The numerical answer to our question of course is 42 and this can also be stored as a variable(var) with the indentifier 'myAnswer' enabling us to call each peice of data wherever it is needed.


I Declare Thee... Variable!

In order to use a variable first we must declare it. This means we tell the console that:

  1. We are declaring a variable using the keyword [ var ].
  2. We are giving it an indentifier for example [ myName ].

We can then assign the [ var ] [ myName ] a string using

[ myName = "Ted"; ]

Of course all of this can be done on one line. It should look something like this:
[ var myName = "Ted"; ]

Let's consider changing our name. Fred is much better than Ted. To change the value held inside a variable, we can use it's identifier [ myname ]to assign [ = ] the new value ["fred"]. Always ending with a semi-colon [ ; ]

[ myName = "Fred"; ]

Using a variable.

It couldn't be simpler to use the data stored in a variable. Wherever you need it, just use it's identifier. For Example:

[ ("Hello world. My name is " + myName); ]
