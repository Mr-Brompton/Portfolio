Literal and Identifiers

Their are two ways of referencing data in JavaScript. There is Literal data - Typed directly into JavaScript and Data Identifiers - which are names that act as a reference for the data it is assigned to.

"Matt"
30
true
null
are all examples of Literal Data.

myName = "Matt";
myAge = 30;
myAim = true;
myBrain = null;
are examples of how to declare identifiers. Once an identifier is declared it can be used in place of the Literal Data.

In JavaScript Identifiers must not use reserved words. The first character of an Identifier must be a letter [A], underscore [_] or a dollar [$] while any subsequent characters may also include numbers.
